Project Portfolio

1. Biodiversity Website

ELOUERA is a website that describes the Biodiversity in Bangka Belitung, identifies Land Cover and predicts Sea Surface Temperature. Proyek ini bertujuan menunjukan interaksi berbagai makhluk hidup, serta hubungan mereka dalam beradaptasi dengan lingkungannya. Keanekaragaman hayati memiliki 3 tingkatan, yaitu keanekaragaman ekosistem, spesies (jenis) dan genetis. Keterkaitan 3 hal tersebut menjadi ciri khas tersendiri didalam keanekaragaman.

2. Teknologi yang Digunakan

  • Visual Studio Code
  • Google Earth Engine
  • Bootstrap
  • Program Language (HTML, CSS)

    • 3. Fitur Utama

    Statistik Umum:

  • Menampilkan Keanekaragaman Hayati yang terdapat di Pulau Bangka Belitung.
  • Melakukan klasifikasi tutupan lahan di Pulau Bangka Belitung untuk menyediakan informasi yang berguna untuk keperluan pemodelan.
  • Menampilkan validasi akurasi terhadap tutupan lahan yang dibuat menggunakan Machine Learning
  • Menampilkan map dan grafik terhadap suhu permukaan laut di Pulau Bangka Belitung.

    • Tujuan Proyek:
  • Menyediakan informasi tentang flora dan fauna khas Pulau Bangka Belitung.
  • Menggunakan Google Earth Engine untuk mengklasifikasikan perubahan tutupan lahan.
  • Melakukan validasi data untuk memastikan keakuratan klasifikasi tutupan lahan dan informasi biodiversitas.
  • Menyediakan peta interaktif untuk membantu visualisasi keanekaragaman hayati dan perubahan lahan.
  • Menampilkan grafik suhu permukaan laut untuk memahami dampak perubahan iklim terhadap ekosistem laut.

    • 4. Sumber Data

  • Flora dan Fauna Khas Pulau Bangka Belitung
